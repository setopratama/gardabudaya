# Gardabudaya

Proyek Firebase untuk Gardabudaya.

## Setup

1. Instal dependensi:
   npm install

2. Inisialisasi Firebase (jika diperlukan):
   firebase login
   firebase init

3. Jalankan server pengembangan:
   npm start

## Pengaturan Docker

Bangun dan jalankan dengan Docker Compose:

```bash
docker compose up --build
```

Aplikasi akan tersedia di http://localhost:5000
